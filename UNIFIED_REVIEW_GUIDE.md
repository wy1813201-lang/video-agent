# 统一审查系统使用指南

## 📌 概述

统一审查系统提供了一个集成的 Web 界面，用于审查 AI 短剧生成过程中的所有关键内容：
- 👥 **人物母版**：角色形象和描述
- 🎥 **视频片段**：各个镜头的生成进度和效果
- ✨ **最终视频**：完整的成片效果

## 🎨 界面特点

### 1. 统一的标签页设计
- 三个标签页切换：人物母版、视频片段、最终视频
- 实时显示每个类别的数量
- 流畅的切换动画

### 2. 人物母版审查
- 卡片式展示所有角色
- 显示角色图片、名称、描述
- 悬停效果突出显示
- 支持多角色并排对比

### 3. 视频片段审查
- 网格布局展示所有视频片段
- 内嵌视频播放器，可直接预览
- 显示镜头ID和生成状态
- 状态标签：已完成/生成中/待生成

### 4. 最终视频审查
- 大屏播放器展示最终成片
- 自动播放功能
- 显示视频元数据：
  - 时长
  - 分辨率
  - 文件大小
  - 帧率

### 5. 审查反馈
- 文本框输入审查意见
- 两个按钮：通过 / 不通过
- 提交后自动关闭

## 🚀 使用方式

### 方式 1：自动触发（推荐）

在生成流程中，系统会在关键节点自动弹出审查界面：

```python
# 在 main.py 中集成
from src.unified_review_system import MetaDirectorWithReview

# 初始化
meta_director = MetaDirectorWithReview(config)

# 1. 人物母版生成后自动审查
if meta_director.review_characters(characters):
    print("✓ 人物母版通过审查")
else:
    print("❌ 人物母版未通过，需要重新生成")

# 2. 视频片段生成后自动审查
if meta_director.review_videos(videos):
    print("✓ 视频片段通过审查")
else:
    print("❌ 视频片段未通过")

# 3. 最终视频生成后自动审查
if meta_director.review_final_video(video_path, metadata):
    print("✓ 最终视频通过审查")
else:
    print("❌ 最终视频未通过")
```

### 方式 2：统一审查所有内容

在流程结束时，一次性审查所有内容：

```python
# 统一审查
approved = meta_director.review_all(
    characters=characters,
    videos=videos,
    final_video=final_video_path
)

if approved:
    print("✓ 所有内容通过审查")
else:
    print("❌ 需要修改")
```

## 🎯 审查要点

### 人物母版审查
检查项：
- ✅ 角色形象是否符合设定
- ✅ 角色特征是否清晰
- ✅ 角色之间是否有明显区分
- ✅ 图片质量是否达标

### 视频片段审查
检查项：
- ✅ 镜头内容是否符合剧本
- ✅ 角色形象是否一致
- ✅ 画面质量是否清晰
- ✅ 镜头衔接是否流畅

### 最终视频审查
检查项：
- ✅ 整体节奏是否合理
- ✅ 剧情是否连贯
- ✅ 音效配乐是否合适
- ✅ 字幕是否准确
- ✅ 时长是否符合要求（50-70秒）

## 💡 使用建议

### 1. 审查时机
- **人物母版**：在生成所有视频前审查，避免浪费资源
- **视频片段**：在合成前审查，确保每个镜头质量
- **最终视频**：最后审查，确保整体效果

### 2. 反馈建议
提供具体的反馈意见，例如：
- ✅ "女主角色形象符合预期，男主需要更冷酷的表情"
- ✅ "第3个镜头画面模糊，需要重新生成"
- ✅ "整体节奏偏慢，建议加快剪辑速度"

避免模糊的反馈：
- ❌ "不好"
- ❌ "有问题"
- ❌ "重做"

### 3. 审查标准
建议制定明确的审查标准：
- 画面质量：清晰度、色彩、构图
- 角色一致性：形象、服装、表情
- 剧情连贯性：逻辑、节奏、情绪
- 技术指标：时长、分辨率、帧率

## 🔧 配置选项

在 `config.yaml` 中配置：

```yaml
meta_director:
  enabled: true
  enable_web_review: true  # 启用 Web 审查界面
  review_port: 5001        # Web 服务器端口
  auto_open_browser: true  # 自动打开浏览器
```
